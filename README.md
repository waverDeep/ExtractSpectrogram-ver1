# ExtractSpectrogram-ver1
다양한 기법을 활용하여 spectrogram, melspectrogram, log melspectrogram, power melspectrogram, MFCC 를 추출하는 스크립트

## Libraries
- librosa : 0.7.1
- numba : 0.48.0
- librosa.display
- matplotlib.pyplot
- scipy
- scipy.io
- scipy.io.wavfile
- scipy.fftpack

## Visualizing
스펙트로그램을 추출한 후 matplotlib으로 시각화까지 진행할 수 있음
